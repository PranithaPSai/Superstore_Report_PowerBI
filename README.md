# PowerBI Dashboard Project : Superstore Analysis
This repository contains the Power BI Dashboard project that I created using the Global Superstore dataset. The dashboard provides in-depth analysis and visualizations to gain insights into various aspects of the business.

## Project Overview
The Superstore Power BI Dashboard is a comprehensive business intelligence project designed to analyze global retail data and uncover valuable insights. Using the Superstore dataset, this dashboard provides a clear picture of business performance across sales, profit, customers, and regions.The goal of the project is to transform raw sales data into meaningful visual insights that help stakeholders make data-driven decisions. By leveraging Power BIâ€™s data modeling, DAX calculations, and interactive visualizations, the report highlights key trends, identifies top-performing segments, and monitors delivery performance across regions and time periods.

## Tech Stack
The Dashboard was built using the following tools and technologies:
* Microsoft Power BI Desktop-Main tool used to build interactive dashboards and reports.
* Power Query Editor-Used for cleaning, shaping, and transforming the raw Superstore dataset.
* DAX (Data Analysis Expressions)-Used to create calculated columns, measures, KPIs, and custom metrics.
* Power BI Map Visuals-Enabled visualization of regional and market-level insights across countries.
* File Format- .pbix for development and .png for dashboard previews.

## Key Features
The Superstore Power BI Dashboard delivers a full analytical view of business performance through intuitive, interactive, and data-driven visualizations.
1) Data Integration: Connected and imported the Global Superstore dataset into Power BI for seamless analysis.
2) Data Cleaning: Transformed and standardized data using Power Query Editor for accuracy and consistency.
3) Data Modeling: Built relationships and DAX measures to enable dynamic and insightful reporting.
4) Interactive Dashboard: Created user-friendly visuals with filters, slicers, and drill-through capabilities.
5) KPI Tracking: Displayed key metrics like sales, profit, quantity, and average delivery days.
6) Regional Analysis: Used map visuals to compare sales and profit across countries and markets.
7) Customer Insights: Identified top profit-generating customers.
8) Market Segmentation: Analyzed performance by region, category, and customer segment.
9) Time Intelligence: Added year slicer for trend and comparative analysis.
10) Visual Design: Focused on clarity, interactivity, and easy navigation for better storytelling.
